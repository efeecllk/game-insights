## Prerequisites

Before starting these tutorials, make sure you have:

1. **Game Insights running locally** - Follow the [Quick Start Guide](/docs/getting-started/quickstart) if you haven't set it up yet
2. **Sample data** (optional) - The application includes demo data, but you can use your own CSV or JSON files
3. **OpenAI API key** (optional) - For AI-powered column detection. Go to Settings to configure this

## How to Follow Along

Each tutorial includes:

- **Step-by-step instructions** with numbered steps
- **What you should see** sections describing expected results
- **Troubleshooting tips** for common issues
- **Next steps** suggestions for further learning

### Conventions Used

Throughout these tutorials:

- `Click [Button Name]` - Click the named button or link
- `Navigate to /path` - Go to that URL path in your browser
- `Enter "value"` - Type the specified text
- **Bold text** - UI element names
- `Code text` - File names, values, or commands
