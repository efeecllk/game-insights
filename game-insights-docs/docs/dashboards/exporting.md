# Export Capabilities

Game Insights provides comprehensive export functionality for dashboards and data in multiple formats.

## Export Formats

| Format | Use Case | Contents |
|--------|----------|----------|
| **PNG** | Presentations, sharing | Visual snapshot |
| **PDF** | Reports, documentation | Multi-page document |
| **JSON** | Backup, version control | Configuration data |
| **CSV** | Analysis, spreadsheets | Raw data |

## Export to PNG

Create high-quality images of your dashboards.

### Options
- **Scale** - 1x, 2x, or 3x resolution
- **Background** - Light or dark theme
- **Include header** - Dashboard title and date

### How to Export
1. Open your dashboard
2. Click **Export** > **PNG**
3. Configure options
4. Click **Download**

## Export to PDF

Generate professional reports.

### Options
- **Page size** - A4, Letter, Legal
- **Orientation** - Portrait or Landscape
- **Include cover** - Title page
- **Include insights** - AI summary page

### PDF Structure
1. Cover page with metadata
2. Dashboard visualization
3. AI insights (optional)
4. Page numbers

## Export to JSON

Save dashboard configuration for backup or sharing.

### Export Types
- **Config only** - Layout and widget settings
- **With data** - Config + current values
- **Full export** - Config + data + history

### Use Cases
- Backup configurations
- Version control
- Share with team
- Template creation

## CSV Data Export

Export raw data from widgets.

### Widget-Level Export
1. Click widget menu (...)
2. Select **Export Data**
3. Download CSV

### Dashboard-Level Export
1. Click **Export** > **CSV**
2. Select widgets to include
3. Download ZIP with multiple CSVs

### CSV Options
- Delimiter (comma, semicolon, tab)
- Include headers
- Date format
- Number format

## Sharing Dashboards

### Share Link
Generate read-only links:
1. Click **Share**
2. Set expiration (optional)
3. Copy and share URL

### Scheduled Reports
Automate email delivery:
1. Click **Schedule Report**
2. Set recipients
3. Choose frequency (daily, weekly, monthly)
4. Select format (PDF, PNG, link)

## Best Practices

### For Presentations
- Use 2x or 3x PNG for clarity
- Consider light theme for projection
- Remove clutter before export

### For Reports
- Use PDF with cover page
- Include date ranges prominently
- Add AI insights for context

### For Data Analysis
- Use CSV for spreadsheet tools
- Export raw data when possible
- Include metadata for context
