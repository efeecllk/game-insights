# Template Marketplace

The Template Marketplace provides a library of pre-built dashboards and funnel configurations created by the community and Game Insights team.

## Overview

Templates accelerate your analytics setup:

- **Pre-configured dashboards** - Ready-to-use layouts
- **Funnel templates** - Common conversion paths
- **Alert rule sets** - Best-practice monitoring
- **Game-type optimized** - Tailored for each genre

## Browsing Templates

1. Navigate to **Dashboard Builder**
2. Click **+ New Dashboard**
3. Select **Browse Templates**

Or access directly from **Templates** in the sidebar.

## Categories

| Category | Focus Area |
|----------|------------|
| **Retention** | Cohort analysis, day-N tracking |
| **Monetization** | Revenue, ARPU, spender tiers |
| **Engagement** | Sessions, DAU, feature usage |
| **Acquisition** | Installs, sources, campaigns |
| **Funnels** | Conversion paths |
| **Alert Sets** | Monitoring rules |

## Game Type Filters

Templates optimized for:

- **Puzzle** - Level progression, boosters
- **Idle** - Prestige cycles, offline time
- **Battle Royale** - Match outcomes, ranks
- **Match-3 Meta** - Story progression, decorations
- **Gacha RPG** - Banner performance, spender tiers

## Importing Templates

1. Click **Preview** to see the template
2. Review included widgets and data requirements
3. Click **Import**
4. Choose import option:
   - **Import as new** - Creates new dashboard
   - **Merge** - Adds to existing dashboard
5. Customize as needed

## Creating Your Own

Share your dashboards:

1. Open your dashboard in Builder
2. Click menu > **Create Template**
3. Fill in details:
   - Name and description
   - Category and game types
   - Tags for discoverability
4. Choose visibility (private/public)
5. Submit for listing

## Template Features

Each template includes:
- Preview image
- Widget list
- Data requirements
- Author and ratings
- Import count

## Best Practices

1. **Preview first** - Check data requirements match yours
2. **Customize after import** - Adjust to your needs
3. **Rate templates** - Help the community
4. **Share yours** - Contribute back
